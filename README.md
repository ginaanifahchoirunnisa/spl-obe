# spl-obe
Solving the linier equation problems using elementary line operations (Gauss and Gauss-Jordan Elimination)

## Program Structure
1. Eliminasi Gauss
2. Eliminasi Gauss-Jordan
3. Metode Matriks Balikan
4. Kaidah Cramer
5. Keluar

## Compiling Program
```
javac *.java
```

## Running Program
```
java Main
```
